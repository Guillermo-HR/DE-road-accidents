## About The Project
The aim of the project is to develop a data pipeline that extracts data from various sources related to road accidents in Mexico. The project proposes to use tools for data governance, data quality, transformation, and storage. The final goal is to create a reliable and efficient data pipeline that allows for the analysis and visualization of road accident data in Mexico.

### Built With

- [Docker](https://www.docker.com/)
- [Apache spark](https://spark.apache.org/)
